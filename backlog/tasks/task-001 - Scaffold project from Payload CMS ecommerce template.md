# Scaffold project from Payload CMS ecommerce template

Bootstrap the project using the official Payload CMS ecommerce template, which scaffolds products, cart, orders, guest checkout, and tests out of the box.

## Steps

- Run `npx create-payload-app@latest` with the ecommerce template
- Verify Next.js App Router is configured
- Pin `@payloadcms/plugin-ecommerce` to the current beta version
- Confirm the dev server runs locally
- Commit the initial scaffold to the repo
