# Trust badges near checkout

Small visual signals that reassure buyers before they complete a purchase.

## Badges to include

- Secure payment (lock icon)
- Satisfaction guarantee
- Free returns (if applicable)

## Steps

- Design or source SVG/icon trust badge components
- Place on: cart page, checkout payment step, and product detail pages
- Keep subtle — these should reinforce confidence without cluttering the layout
- RTL-compatible layout
